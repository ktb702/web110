HTML ELEMENTS

Element Description Block, Inline or N/A Container or Empty
<!-- --> Comments are used to provide instructions and explanations. They are not displayed in the browser. N/A Container
<!DOCTYPE> HTML specification telling a browser which W3 rules the page is written in. N/A Empty
<a> Specifies a hyperlink, which will link from one page to another page. Inline Container
<article> Specifies standalone content. Block Container
<aside> Defines content aside (close to) the content that it is placed by. Block Container
<body> Contains all contents of an HTML document. Block Container
<br> Inserts a single line break. Inline Empty
<dd> Describes a name in a description list. Block Container
<div> Defines a division or section of the document. Block Container
<dl> Defines a description list. Block Container
<dt> Defines a term in a description list. Block Container
<em> It renders emphasized text. The text output will be in italics. Inline Container
<fieldset> It draws a box around related elements in a form. Block Container
<figcaption> Defines a caption for the figure element. Block Container
<figure> Specifies self contained content like illustrations or diagrams. Block Container
<footer> Defines a footer for a document or section. Block Container
<form> Creates an HTML form for user input. Block Container
<h1>to<h6> Defines HTML headings in order of descending importance. Block Container
<head> Can include a title for a document, scripts, styles, meta tags, and more. Block Container
<header> Represents a container for introductory content or a set of navigational links. Block Container
<html> Tells the document that this is an HTML document. N/A Container
<iframe> Specifies an inline frame. Inline Container
<img> Creates a holding space for the referenced image that is linked to the HTML page. Inline Empty
<input> Specifies an input field where the user can enter data. Inline Empty
<label> Provides a usability improvement for mouse users. Inline Container
<li> Defines a list item. Block Container
<link> Defines a link between a doument and an external source. N/A Empty
<meta> Typically used to specify page description, keywords, author, or other data. It will not be displayed on the page. N/A Empty
<nav> Defines a set of navigation links. Block Container
<ol> Defines an ordered (numbered) list. Block Container
<p> Defines a paragraph and automatically adds some space before and after the element. Block Container
<script> Used to define client-side script (JavaScript). Block Container
<section> Defines sections within a document such as chapters, headers, or footers. Block Container
<strong> It is a phrase tag that defines important text. Inline Container
<style> Defines style information of an HTML document. Block Container
<table> Defines an HTML table. Block Container
<td> Defines a standard cell of data in an HTML table. Block Container
<th> Defines a header cell in an HTML table. Block Container
<title> Defines the title of a document. This tag is required. Block Container
<tr> Defines a row in an HTML table. Block Container
<ul> Defines an unordered (bulleted) list. Block Container
